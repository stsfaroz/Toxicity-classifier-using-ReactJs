# Toxicity classifier using ReactJs

 ![React App](https://github.com/stsfaroz/Toxicity-classifier-using-ReactJs/blob/main/ret.png) 
 
 <img src="https://github.com/stsfaroz/Toxicity-classifier-using-ReactJs/blob/main/ret.png" data-canonical-src="https://gyazo.com/eb5c5741b6a9a16c692170a41a49c858.png" width="200" height="400" />
 
 This is an application for classifying the Toxicity in text based on the below categories.
* toxic
* evere_toxic
* obscene
* threat
* insult
* identity_hate

> My setup 
```
$ node --version
v10.14.2
$ npm --version
6.4.1
```
## Getting Started

Download the repository 
In the project directory, you can run:

###### `$ npm install`

###### `$ npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
